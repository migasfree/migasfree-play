# User How-to Guides

This document provides step-by-step instructions for common user tasks in **Migasfree Play**.

> **Note**: For a description of the icons and UI elements, see [Technical Reference](../reference/technical.md). If you encounter errors, see [Troubleshooting Guide](troubleshooting.md).

---

## ⚙️ How to configure the application

1. **Server Configuration**: The application automatically reads the server URL from `/etc/migasfree.conf`. Ensure this file is correctly configured by your administrator.
2. **User Credentials**: Ensure the `MFP_USER` and `MFP_PASSWORD` environment variables are set correctly for your environment.

## 📦 How to manage applications

1. **Find an app**: Click the **Apps** icon (grid) in the header and use the search bar or categories.
2. **Install**: Click the **green arrow** button on an application's card.
3. **Uninstall**: Click the **red trash bin** button.

## 🖥️ How to manage devices

1. **View inventory**: Click the **Devices** icon (printer) to see tracked hardware.
2. **Add a device**: If enabled, use the registration interface to add new peripherals to your computer.

## 🔄 How to synchronize with the server

Synchronization ensures your computer has the latest configuration and security updates.

1. Click the **Floating Action Button** (circular "play" button) in the bottom-right corner.
2. The process runs in the background.
3. Once finished, verify the status in the **Details** section.

## 📜 How to view execution logs

1. Click the **Details** icon (list) to see the history of actions.
2. Click on any entry to see the full console output and start/end times.
